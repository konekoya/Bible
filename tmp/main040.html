關於離線聖經系統

這個系統是用 Javascript 寫的，因此它有幾個特性，

  1. 要在瀏覽器中執行。
  2. 一開始載入的時間比較長，因為需要將聖經的經文全部載入。
  3. 搜尋的速度比較慢，因為 Javascript 的執行速度有點慢。

0.40版主要的改變有：

  1. 將 frame 拿掉。前陣子才發現在我自己的筆記型電腦裡面，0.10~0.30 都無法使用 IE 來執行，我猜是 frame 的緣故，一直都改不出來，因此才想要改一個不需要用到 frame 的版本出來，現在終於改出來了，只是尚未在我的筆記型電腦試過，今天回家時再來測試看看。萬一您已經習慣有 frame 的方式，只好請您繼續使用舊版，我以後會儘可能不使用 frame。
  2. 所找的字串會顯現出不同的顏色，這是本來就有的功能，這次將這個顏色延續到進入每一章之後。例如：搜尋「恩典 憐憫」會找到 8 節經文，其中第一節是出埃及記第 34 章，點進出埃及記 34 章之後，有出現「恩典」與「憐憫」的地方仍會出現紅色，持續按前一章、後一章時，這個功能仍會持續。可以用「神」或「耶和華」試看看。

最近我對 Javascript 愈來愈有興趣，研究將程式的結果顯現在同一個檔案也有另一個目的，就是準備將一些程式的題目的答案加到我的題庫中，用 Javascript 將程式寫出來，直接放在網站上執行，我想效果應該是比較即時的。另一個目的是希望更了解 AJAX，我想 AJAX 的主要程式還是 Javascript，等我對 Javascript 更熟悉的時候，就要來研究 AJAX，或許以後會改一份 AJAX 的聖經查詢系統出來也說不定。
